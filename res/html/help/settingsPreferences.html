Settings - Preferences

Debug Output

Toggle the display of a debug console that shows real-time operations being executed.

Auto Zip Topic

Automatically zips and republishes a topic upon any modification. Publishing time depends on the topic's size.

Allow Gossip

Allow your node to share its connections with others. For example, if A is connected to B and B is connected to C, enabling gossip can connect A and C automatically.

Go Public

Converts your Redlines node into a publicly accessible one. Attempts to use UPnP to open your port; otherwise, you'll need to set up port forwarding manually. (Private and public topics are separateâ€”local-only nodes cannot access public topics and vice versa.)

'Ding!' on Annotation

Plays a 'Ding!' sound when an annotation is received on an open topic.
