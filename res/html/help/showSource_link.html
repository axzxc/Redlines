Show Source - Link (Author)

Show Raw

Toggles the display of the raw HTML representation of the web link, allowing you to inspect its structure and metadata. Useful for debugging or manual edits.

Open

Launches the referenced web link in your system's default browser. Note: If no compatible browser is configured or the link is malformed, this may fail.

Download

Downloads the raw HTML file for the linked page to a location of your choice. This does not include external assets (e.g., CSS, scripts) unless manually fetched.

Edit

Allows you to modify the link's description text within your source list. This is helpful if you want to clarify what the link is about or correct any typos.

Delete

Permanently removes the link source from your collection. This action cannot be undone, so be sure before confirming.
