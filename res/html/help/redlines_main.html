Redlines - Main Interface Overview

Node Search Bar

Input key words and search all your nodes to see whether they have any topics containing them.

Start Topic Buttons

You can initiate a new research thread using one of two buttons:

  • Start Topic with a Primary Source: For firsthand evidence (e.g. documents, interviews, original media).
  • Start Topic with a Secondary Source: For analysis or interpretation (e.g. books, articles, commentary).

Saved Topics

This section shows topics you've already started. Private topics will be indicated by the locked icon while public ones will have a globe icon.
