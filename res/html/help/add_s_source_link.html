Add Link

Paste the URL of any publicly accessible website—such as an article, blog post, or documentation page—as a source. Redlines will attempt to capture a full offline view using a headless browser, preserving the content even if the site later changes or goes offline.

Source Description

Write a short description that explains what the linked website is and how it supports your topic. This helps readers quickly assess its relevance before opening the full archive.
